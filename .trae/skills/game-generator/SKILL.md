# 游戏生成器

## 功能
- 生成时兴、清爽风格的小游戏
- 支持多种游戏类型：休闲、益智、动作等
- 自动创建完整的游戏项目结构
- 包含游戏核心逻辑、UI界面和基础玩法

## 游戏特点
- **清爽风格**：采用简约、现代的设计风格
- **真实感**：流畅的动画效果和响应式交互
- **时兴元素**：融合当前流行的游戏玩法和设计趋势
- **跨设备兼容**：支持桌面端和移动端

## 生成的项目结构
```
game/
├── index.html          # 游戏主页面
├── css/style.css       # 样式文件
├── js/game.js          # 游戏核心逻辑
└── assets/             # 游戏资源
    ├── images/         # 图片资源
    └── sounds/         # 音效资源
```

## 使用方法
1. 用户触发："开始，生成一个时兴小游戏"
2. 技能自动执行：创建游戏项目结构
3. 生成完整的游戏代码和资源
4. 测试游戏运行效果

## 游戏类型
- 休闲类：消除、收集、养成
- 益智类：解谜、推理、策略
- 动作类：跑酷、射击、平台

## 技术栈
- HTML5 Canvas / DOM
- CSS3 动画
- JavaScript ES6+
- 响应式设计

## 交付标准
- 游戏可直接在浏览器中运行
- 完整的游戏玩法和关卡设计
- 美观的视觉效果和流畅的交互体验
- 代码结构清晰，易于维护和扩展
