# Neon Cyberpunk Skill

霓虹赛博朋克风格设计，创建未来科技感、霓虹灯光效果。

## 设计理念

### 核心美学
- **未来主义**: 科幻、赛博朋克的视觉风格
- **霓虹灯光**: 发光效果、霓虹色彩
- **科技感**: 数字化、网格、扫描线
- **暗黑基调**: 深色背景配霓虹点缀

## 色彩系统

### 霓虹色彩
- **霓虹粉** (#ff006e): 主要强调色
- **霓虹蓝** (#00f5ff): 科技感主色
- **霓虹紫** (#b537f2): 神秘感配色
- **霓虹黄** (#ffd60a): 警告/高亮色

### 背景色
- **深色背景** (#0a0e27): 主背景
- **卡片背景** (#141936): 组件背景
- **渐变叠加**: 紫蓝渐变

### 文字色
- **主文字**: 白色 (#ffffff)
- **次要文字**: 霓虹蓝 (#00f5ff)
- **强调文字**: 霓虹粉 (#ff006e)

## 视觉效果

### 霓虹发光
```css
.neon-text {
    color: #00f5ff;
    text-shadow: 
        0 0 10px #00f5ff,
        0 0 20px #00f5ff,
        0 0 40px #00f5ff;
}

.neon-box {
    border: 2px solid #00f5ff;
    box-shadow: 
        0 0 10px rgba(0, 245, 255, 0.3),
        inset 0 0 20px rgba(0, 245, 255, 0.1);
}
```

### 扫描线效果
```css
.scanline {
    position: fixed;
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    background: linear-gradient(
        to bottom,
        transparent 50%,
        rgba(0, 0, 0, 0.1) 50%
    );
    background-size: 100% 4px;
    pointer-events: none;
    opacity: 0.3;
}
```

### 网格背景

## 动效设计

### 故障动画 (Glitch)
```css
@keyframes glitch {
    0%, 90%, 100% {
        text-shadow: 
            0 0 10px #00f5ff,
            0 0 20px #00f5ff;
    }
    92% {
        text-shadow: 
            -2px 0 #ff006e,
            2px 0 #00f5ff;
    }
    94% {
        text-shadow: 
            2px 0 #ff006e,
            -2px 0 #00f5ff;
    }
}
```

### 闪烁效果
```css
@keyframes blink {
    0%, 50%, 100% { opacity: 1; }

}
```


## 最佳实践

### 性能优化
- 限制发光效果数量
- 使用 transform 而非 position
- 优化动画帧率
- 避免过度使用模糊

### 可访问性
- 确保文字可读性
- 提供高对比度选项
- 避免纯光效导航
- 键盘导航支持

### 响应式设计
- 移动端简化效果
- 调整发光强度
- 优化动画性能
- 适配不同屏幕

## 禁止事项

- ❌ 过度使用发光效果（影响性能和可读性）
- ❌ 忽视文字对比度
- ❌ 过于复杂的背景
- ❌ 过多的动画同时运行
- ❌ 忽视移动端性能
- ❌ 缺乏视觉层级
