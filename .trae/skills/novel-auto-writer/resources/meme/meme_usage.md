# Meme Usage Guide

## Meme Usage Principles

1. **Context Appropriateness**: Use memes that fit the scene context
2. **Freshness > Quantity**: Prefer newer memes over stale ones
3. **Diversity**: Mix different categories of memes
4. **Alternatives Ready**: Have backup memes for expired ones
5. **Reverse Usage**: Use reverse examples to avoid clichés

## Per-Chapter Requirements

| Element | Minimum | Recommended |
|:---|:---:|:---:|
| New/Hot Memes | 5 | 8+ |
| Popular Element Combos | 1 | 2+ |
| Anti-Trope Designs | 1 | 2+ |

## Meme Integration Styles

### Style 1: Inline Commentary
```
主角一拳轰出，"欧拉欧拉欧拉！"【弹幕：欧拉连打！】
```

### Style 2: Bracket Thoughts
```
反派：（自以为天下无敌）
主角：（呵，究极生物？）
```

### Style 3: Breaking the Fourth Wall
```
作者：这时候应该来个反转...
读者：来了来了！
```

### Style 4: Meme Mashup
```
"龟派气功波！"
"就这？我可是有光之人！"
【弹幕：迪迦？？？】
```

## Meme Freshness Check

Before using a meme, check:
1. Is it from the current year or last year?
2. Is trending_level A or B?
3. Have alternatives been provided?
4. Does it fit the scene?

## Trending Level Usage

| Level | Usage Frequency | Context |
|:---:|:---:|:---|
| A | Every chapter | Main meme |
| B | 2-3 chapters | Support meme |
| C | 1 chapter | Occasional |
| D | Reference only | Background |

## Monthly Update Checklist

- [ ] Check trending_level for all memes
- [ ] Add new memes from current events
- [ ] Mark expired memes with alternatives
- [ ] Archive memes with use_count = 0 for 1+ year
- [ ] Review reader feedback for meme effectiveness
