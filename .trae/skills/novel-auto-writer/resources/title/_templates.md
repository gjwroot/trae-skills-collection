# Title Templates

## Title Style Categories

### 1. Shock Style (震惊体)

```
Pattern: 震惊!XXX竟然XXX
Examples:
- 震惊!废物女婿竟然是隐藏大佬
- 震惊!我随手救的女孩竟是大小姐
- 震惊!平凡少年体内封印着绝世血脉
```

### 2. Twist Style (反转体)

```
Pattern: 以为XXX，结果XXX
Examples:
- 以为只是普通高考，没想到是修仙入门试
- 以为被退婚是耻辱，没想到未婚妻是废柴
- 以为在底层挣扎，没想到是气运之子
```

### 3. Suspense Style (悬疑体)

```
Pattern: XXX的真相令人XXX
Examples:
- 父母失踪的真相令人震惊
- 体内封印的真相令人窒息
- 身份背后的真相令人发指
```

### 4. Hot Word Style (热词体)

```
Pattern: XXX才是真正的XXX
Examples:
- 重生才是真正的穿越者福利
- 低调才是真正的强者姿态
- 苟住才是真正的生存之道
```

### 5. Meme Style (梗文体)

```
Pattern: XXX (meme combination)
Examples:
- 开挂重生:这一世我不会再输
- 战神归来:老婆女儿快出来看
- 歪嘴龙王:赘婿的身份瞒不住了
```

### 6. Question Style (疑问体)

```
Pattern: XXX?XXX!
Examples:
- 废物?看我一拳碎苍穹!
- 退婚?抱歉我早有婚约在身!
- 穷?不好意思我是隐世家族继承人!
```

### 7. Challenge Style (挑战体)

```
Pattern: 别说XXX，XXX一样XXX
Examples:
- 别说我是废物，考编一样秒杀天才
- 别说我是赘婿，经商一样富可敌国
- 别说我是新人，打仗一样横扫千军
```

### 8. Truth Style (打脸体)

```
Pattern: 不是说XXX吗?XXX!
Examples:
- 不是说我是废物吗?怎么全网找我拜师!
- 不是说我是穷鬼吗?怎么银行卡余额为零!
- 不是说我是舔狗吗?怎么妹妹倒追我!
```

## Title Structure

### Novel Title Format
```
《主题词》 或 《背景+主题》
Examples:
- 《九州问道录》
- 《都市修仙录》
- 《重生之我在仙界当奶爸》
- 《全球高武:我的功法能自创》
```

### Chapter Title Format
```
第X章 主标题·副标题（可选）
Length: 8-16 characters
Separator: · (recommended)

Examples:
- 第1章 觉醒
- 第1章 觉醒·命运的转折
- 第1章 混沌之始·天地未分
```
