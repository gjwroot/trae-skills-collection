# Fanqie (番茄小说) Title Library

## 番茄风Title特点

- 直白、接地气、爽感强
- 多用热门网络词汇
- 强调反差和打脸
- 字数通常在12-20字
- 标题即内容概要

## 番茄Title公式

### 新媒体公式
```
[身份]+[状态]+[结果]
Examples:
- 身份：妈咪、夫人、总裁、千金、萌宝
- 状态：掉马、翻身、逆袭、打脸、团宠
- 结果：宠上天、嫁入豪门、成了团宠
```

### 战神公式
```
[类型]+[身份]+[老婆/老公]+[设定]
Examples:
- 类型：战神、神医、狂婿、天师、兵王
- 身份：归来、出狱、重生、隐世
- 老婆：冰山总裁、美女总裁、小穷鬼、母老虎
```

### 重生公式
```
[类型]+[结果]+[对象]
Examples:
- 类型：重生、穿书、系统、甜宠
- 结果：打脸、翻身、逆袭、宠上天
- 对象：死对头、团宠、千金、萌宝
```
