# Feilu (飞卢小说) Title Library

## 飞卢风Title特点

- 极度追求"爽点"
- 标题即核心爽点
- 多用夸张词汇
- 强调装逼打脸
- 字数通常在15-25字

## 飞卢Title公式

### 直播流公式
```
[类型]+[场景]+[结果]
Examples:
- 类型：直播、网游、娱乐
- 场景：被撤展、开局、观众看
- 结果：给工资、变强、哭灵
```

### 都市流公式
```
[类型]+[身份/系统]+[打脸/装逼]
Examples:
- 类型：都市、职场、娱乐圈
- 系统：神级选择、打脸就变强
- 打脸：影帝、大小姐、总裁
```

### 洪荒流公式
```
[题材]+[开局]+[逆天操作]
Examples:
- 题材：洪荒、玄幻、仙侠
- 开局：绑架、强拆、召唤
- 逆天：娲皇宫、诸天、神仙
```
