# Jinjiang (晋江文学城) Title Library

## 晋江风Title特点

- 偏向言情、纯爱、悬疑
- 强调情感线和人物关系
- 标题较文艺，富有意境
- 字数通常在8-14字
- 常带有CP信息或情感标签

## 晋江Title公式

### 言情公式
```
[设定]+[人设]+[情感]
Examples:
- 设定：穿书、重生、系统、星际、末世
- 人设：团宠、病娇、软萌、傲娇、大佬
- 情感：甜、虐、宠、撩、酸爽
```

### 纯爱公式
```
[设定]+[CP关系]+[互动]
Examples:
- 设定：校园、电竞、娱乐圈、古代
- CP：竹马、天降、竹马VS天降
- 互动：暗恋、追妻火葬场、先婚后爱
```
