# Qidian (起点中文网) Title Library

## 起点风Title特点

- 多用"震惊"、"无敌"、"逆天"等冲击性词汇
- 强调主角身份反转和打脸情节
- 系统流、修仙流、都市流为主
- 字数通常在10-18字

## 热门Title Pattern

### 战神归来类
```
《战神归来:我有七个姐姐》
《战神奶爸:女儿才是真大佬》
《战神隐市:老婆是冰山总裁》
《战神出狱:那些年欠的债该还了》
《战神重生:这一世我为王》
```

### 系统流
```
《神级选择:我选择继承万亿资产》
《神级签到:开局打卡SSS级功法》
《神级败家系统:花钱就变强》
《神级选择:从摆摊到世界首富》
《神级养殖:养什么都能成神》
```

### 修仙类
```
《玄幻:我师兄太稳健了》
《修仙:资质太高怎么办》
《都市修仙:我真不想吃软饭》

《修仙归来:校花老婆求我下山》
```

### 都市流
```
《都市:我真没想当神医啊》
```

## 起点Title公式

```
[身份/能力]+[场景/对象]+[结果/反转]
Examples:
- 身份：战神、兵王、神医、赘婿、神豪、学霸
- 场景：归来、出狱、重生、隐市、修仙、都市
- 反转：无敌、打脸、逆袭、翻盘、崛起
```
