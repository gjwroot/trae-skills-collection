# Rap &amp; Lyrics Writer

## Overview
A specialized skill for creating rhyming rap lyrics, song lyrics, and rap copywriting.

## When to Use
- When user asks to write rap lyrics
- When user asks to write song lyrics
- When user asks for rhyming content
- When user asks for hip-hop or rap copywriting
- When user wants to create a rap song

## How to Use

### 1. Gather Requirements
Ask the user about:
- Topic/theme of the rap/lyrics
- Style preference (e.g., old school, trap, melodic, boom bap)
- Mood/vibe (e.g., energetic, chill, aggressive, emotional)
- Target language (Chinese, English, etc.)
- Any specific words or phrases they want included

### 2. Writing Guidelines
- **Rhyme Scheme**: Use consistent rhyme patterns (AABB, ABAB, ABBA, etc.)
- **Flow**: Maintain a natural rhythm and cadence
- **Structure**: Typically includes Intro, Verses, Chorus/Hook, Bridge, Outro
- **Wordplay**: Incorporate metaphors, similes, double entendres
- **Authenticity**: Keep the voice genuine to the style

### 3. Output Format
```
[Title]
(Intro)
[Intro lyrics]

(Hook)
[Chorus lyrics]

(Verse 1)
[Verse 1 lyrics]

(Hook)
[Chorus lyrics]

(Verse 2)
[Verse 2 lyrics]

(Hook)
[Chorus lyrics]

(Outro)
[Outro lyrics]
```

## Examples

### Example 1: Motivational Rap
**Prompt**: "Write a motivational rap about chasing dreams"
**Output**: A structured rap with verses, hook, and rhymes about perseverance

### Example 2: Love Song Lyrics
**Prompt**: "Write a love song with a catchy hook"
**Output**: Romantic lyrics with a memorable chorus and emotional verses

### Example 3: Party Anthem
**Prompt**: "Write an upbeat party rap"
**Output**: High-energy lyrics perfect for celebration and dancing

## Tips
- Keep the language natural and conversational
- Match the rhythm to the intended beat
- Use repetition for hooks to make them memorable
- Incorporate the user's personal style or brand
- Adjust complexity based on the user's needs

## Best Practices
- Always ask for clarification if requirements are unclear
- Offer to revise and improve the lyrics
- Provide multiple versions or options when possible
- Maintain professionalism and respect in all content
