## ⚙️ 配置步骤

### 1. 获取 AppSecret

在微信开发者平台：
1. 进入「基础信息」
2. 点击 AppSecret 旁边的「重置」
3. 扫码验证
4. **立即复制**AppSecret（只显示一次！）

### 2. 设置 IP 白名单

在微信开发者平台：
1. 进入「基础信息」→「开发密钥」
2. 点击「设置IP白名单」
3. 添加 IP：`223.104.43.57`
4. 保存

### 3. 存储凭证

#### 方式一：环境变量（推荐）

在 `~/.zshrc` 或 `~/.bashrc` 中添加：

```bash
export WECHAT_MP_APPID="wx502a3f040c0be7a9"
export WECHAT_MP_APPSECRET="effa61ce6dfb91039207337bcd1b25ab"
export WECHAT_MP_ACCOUNT_NAME="分支人生"
```

重载配置：
```bash
source ~/.zshrc
```

#### 方式二：本地配置文件

创建 `~/.wechat-mp/accounts.json`：

```json
{
  "branch_life": {
    "name": "分支人生",
    "appid": "wx502a3f040c0be7a79",
    "appsecret": "你的AppSecret",
    "ip_whitelist": ["223.104.43.57"]
```

## 🆘 常见问题

**Q: AppSecret 忘记了怎么办？**
A: 去开发者平台「重置」，注意重新保存！

**Q: IP 变化了怎么办？**
A: 用 `curl -4 ifconfig.me` 查新 IP，去开发者平台更新白名单

**Q: 提示 40164 错误？**
A: IP 不在白名单，检查白名单配置
