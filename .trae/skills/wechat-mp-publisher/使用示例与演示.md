## 快速开始示例

### 示例 1：发布一篇简单文章

**跟 AI 说：**
```
帮我写一篇关于"如何用 AI 自动化工作流"的文章，然后发到我的公众号
```

**AI 会自动完成：**
1. ✅ 撰写文章内容（Markdown 格式）
2. ✅ 转换为微信兼容的 HTML
3. ✅ 上传所有图片到微信 CDN
4. ✅ 创建公众号草稿
5. ✅ 返回 draft_media_id

**你需要做的：**
1. 登录 mp.weixin.qq.com
2. 进入「草稿箱」
3. 预览文章
4. 点击「发布」

## 常用指令集锦

### 基础发布类

| 指令 | 说明 |
| --- | --- |
| `帮我把这篇文章发到公众号` | 发布当前正在编辑的文章 |
| `把最新的博客文章发布到公众号` | 查找并发布最新博客 |
| `创建一个公众号草稿` | 只创建草稿，不自动发布 |

### 草稿管理类

| 指令 | 说明 |
| --- | --- |
| `列出我最近的公众号草稿` | 查看草稿列表 |
| `删除那个 draft_media_id 为 xxx 的草稿` | 删除指定草稿 |
| `清空所有未使用的草稿` | 批量清理草稿 |

### 多账号类

| 指令 | 说明 |
| --- | --- |
| `帮我把文章发到老婆的品牌号上` | 切换到指定账号 |
| `使用 my-personal-account 账号发布` | 使用账户名切换 |
| `先列出我有哪些公众号账号` | 查看可用账号 |

### 示例文章模板

```markdown
# 标题：AI 时代的工作方式变革

## 副标题：从手动重复到智能自动化

---

## 引言

在这个 AI 飞速发展的时代，我们的工作方式正在经历前所未有的变革。

## 核心观点

### 1. 自动化不是替代，而是赋能

AI 不会取代人类，而是让我们从繁琐的重复劳动中解放出来。

### 2. 技能的重塑

> 「你不需要学会所有技能，你只需要学会如何指挥 AI。」

## 实践案例

### 案例一：公众号自动发布

以前手动发布一篇公众号文章需要：
1. 复制内容
2. 调整格式
3. 上传图片
4. 设置封面

现在只需要一句话：
```
帮我把这篇文章发到公众号
```

## 总结

拥抱 AI，让工作更高效，让生活更美好。

---

*作者：你的名字*
*发布于：2026年*
```

## 多账号切换示例

### 场景：你有两个公众号

1. **个人号** - 分享技术心得
2. **老婆的品牌号** - 推广产品

### 配置凭证（密码管理器示例）

**个人号凭证：**
```
项目名: WeChat MP - 个人号
AppID: wx1234567890abcdef
AppSecret: xxxxxxxxxxxxxxxxxxxxxxxx
账户名: personal
```

**品牌号凭证：**
```
项目名: WeChat MP - 老婆的品牌号
AppID: wx0987654321fedcba
AppSecret: yyyyyyyyyyyyyyyyyyyyyyyy
账户名: wife-brand
```

### 切换账号发布

**发布到个人号：**
```
用 personal 账号，把这篇技术文章发到公众号
```

**发布到品牌号：**
```
帮我把产品推广文章发到老婆的品牌号上
```

## 💡 小贴士

1. **先测试，后发布**
   - 先用简单文章测试流程
   - 确认没问题后再发正式内容

2. **善用草稿**
   - AI 创建的是草稿，你可以随时修改
   - 满意后再手动点击发布

3. **图片优化**
   - 建议封面图尺寸：2.35:1（约 900x383 像素）
   - 文章内图建议宽度不超过 1080 像素

4. **备份重要内容**
   - 发布前先备份 Markdown 原文
   - 保留 draft_media_id 以备需要
