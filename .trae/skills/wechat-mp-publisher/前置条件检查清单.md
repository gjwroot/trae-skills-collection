# ✅ 前置条件检查清单

在开始使用微信公众号自动发布功能前，请逐项检查以下内容：

---

## 第一部分：公众号准备

- [ ] **1.1 拥有微信公众号**
  - 类型：订阅号 或 服务号
  - 状态：已认证（推荐，功能更多）或未认证

- [ ] **1.2 拥有管理员权限**
  - 可以登录 mp.weixin.qq.com
  - 可以登录 developers.weixin.qq.com

---

## 第二部分：开发者凭证

- [ ] **2.1 获取 AppID**
  - [ ] 访问 https://developers.weixin.qq.com/platform
  - [ ] 扫码登录（管理员微信）
  - [ ] 进入「我的业务与服务」→ 找到公众号
  - [ ] 进入「基础信息」→ 记录 AppID

- [ ] **2.2 获取 AppSecret**
  - [ ] 在同一页面点击「启用」或「重置」
  - [ ] **立即复制**AppSecret（只显示一次！）
  - [ ] 保存到密码管理器（Bitwarden/1Password等）

- [ ] **2.3 凭证安全存储**
  - 方案 A：环境变量（推荐单账号）
  - 方案 B：密码管理器（推荐多账号）
  - 方案 C：本地配置文件（⚠️ 不推荐）

---

## 第三部分：网络配置

- [ ] **3.1 查看公网 IP**
  ```bash
  curl -4 ifconfig.me
  ```
  - [ ] 记录当前 IP 地址

- [ ] **3.2 添加 IP 白名单**
  - [ ] 在微信开发者平台进入「接口配置」
  - [ ] 找到「IP 白名单」设置
  - [ ] 添加你的公网 IP
  - [ ] 保存设置

---

## 第四部分：测试验证

- [ ] **4.1 测试获取 Token**
  ```bash
  curl -s -4 "https://api.weixin.qq.com/cgi-bin/token?grant_type=client_credential&appid=YOUR_APPID&secret=YOUR_APPSECRET"
  ```
  - [ ] 成功返回 access_token
  - [ ] 没有报错（40164/40013/40001等）

- [ ] **4.2 测试上传图片（可选）**
  - [ ] 准备一张测试图片 test.jpg
  - [ ] 使用返回的 access_token 测试上传
  - [ ] 成功返回 media_id

---

## 第五部分：AI 工具配置

- [ ] **5.1 技能文件已安装**
  - [ ] SKILL.md 存在于技能目录
  - [ ] skill.yaml 存在

- [ ] **5.2 重启 AI 工具**
  - [ ] 重启 OpenCode / Claude Code
  - [ ] 确认技能已加载

---

```

```

或者：

```
帮我列一下需要完成的配置步骤
```

## 常见问题快速排查

| 问题 | 检查项 |
| --- | --- |
| 40164 错误 | ✅ IP 白名单已更新 |
| 40013 错误 | ✅ AppID 正确 |
| 40001 错误 | ✅ AppSecret 正确 |
| 找不到技能 | ✅ 已重启 AI 工具 |
| 图片不显示 | ✅ 已上传到微信 CDN |
