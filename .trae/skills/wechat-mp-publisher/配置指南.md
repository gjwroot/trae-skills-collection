## 获取 AppID 和 AppSecret

### 2024-2025 最新路径

⚠️ **重要**：微信后台改版，旧路径已失效！

#### 步骤：

1. **打开微信开发者平台**
   - 访问：https://developers.weixin.qq.com/platform
   - 使用管理员微信扫码登录

2. **进入公众号管理**
   - 点击「我的业务与服务」
   - 找到你的公众号卡片
   - 点击进入公众号详情

3. **查看基础信息**
   - 在左侧菜单找到「基础信息」
   - 这里可以看到你的 **AppID**
   - 点击「启用」或「重置」获取 **AppSecret**

#### 常见问题解决

**Q: 开发者平台里看不到我的公众号？**
A: 需要先绑定，但绑定需要 AppID（死循环）
→ **解决方法**：
   1. 登录 mp.weixin.qq.com（旧版后台）
   2. 进入「广告与服务」→「微信支付」
   3. 在商户号管理页面旁边会显示 AppID

**Q: AppSecret 显示「未知状态」？**
A: 需要用管理员身份在开发者平台「基础信息」里点击「启用」

**Q: AppSecret 只显示一次怎么办？**
A: **立即复制并保存**到密码管理器（Bitwarden、1Password 等）

## 设置 IP 白名单

### 为什么需要 IP 白名单？
微信 API 只接受白名单内 IP 的请求，这是安全措施。

### 步骤：

1. **查看当前公网 IP**
   ```bash
   curl -4 ifconfig.me
   ```
   或访问：https://ifconfig.me

2. **添加到白名单**
   - 在微信开发者平台，进入「接口配置」
   - 找到「IP 白名单」设置
   - 把你的公网 IP 添加进去

### ⚠️ 注意事项

- **家庭宽带通常是动态 IP**，IP 变化后会报 40164 错误
- 解决方案：IP 变化后重新执行上面的步骤更新白名单
- **curl 必须用 -4 参数**，确保走 IPv4，不经过代理

## 凭证存储方案

### 方案一：环境变量（推荐用于单账号）

```bash
# 在 ~/.zshrc 或 ~/.bashrc 中添加
export WECHAT_MP_APPID="wx1234567890abcdef"
export WECHAT_MP_APPSECRET="xxxxxxxxxxxxxxxxxxxxxxxx"
```

重新加载配置：
```bash
source ~/.zshrc
```

### 方案二：密码管理器（推荐用于多账号）

使用 Bitwarden、1Password 等，存储格式：

```
项目名: WeChat MP - 我的公众号
用户名: (可选)
密码: (AppSecret)
备注:
  AppID: wx1234567890abcdef
  账户名: my-personal-account
```

### 方案三：本地配置文件（不推荐，不安全）

创建 `~/.wechat-mp/config.json`：
```json
{
  "accounts": {
    "my-personal-account": {
      "appid": "wx1234567890abcdef",
      "appsecret": "xxxxxxxxxxxxxxxxxxxxxxxx"
    },
    "wife-brand-account": {
      "appid": "wx0987654321fedcba",
      "appsecret": "yyyyyyyyyyyyyyyyyyyyyyyy"
    }
  }
}
```

⚠️ **重要**：不要把这个文件提交到 Git！

## 测试验证

### 第一步：测试获取 Token

```bash
# 替换为你的 AppID 和 AppSecret
curl -s -4 "https://api.weixin.qq.com/cgi-bin/token?grant_type=client_credential&appid=YOUR_APPID&secret=YOUR_APPSECRET"
```

**成功响应**：
```json
{
  "access_token": "61_Nk12345...",
  "expires_in": 7200
}
```

**常见错误**：
- `40164`: IP 不在白名单 → 检查 IP 白名单
- `40013`: AppID 无效 → 检查 AppID 是否正确
- `40001`: AppSecret 错误 → 重置 AppSecret

### 第二步：测试上传图片

准备一张测试图片 `test.jpg`，然后：

```bash
# 先获取 access_token（上面的命令）
# 然后替换 ACCESS_TOKEN
curl -s -4 -F "media=@test.jpg" "https://api.weixin.qq.com/cgi-bin/media/upload?access_token=ACCESS_TOKEN&type=image"
```

**成功响应**：
```json
{
  "type": "image",
  "media_id": "MEDIA_ID_123",
  "created_at": 123456789
}
```

## 📞 遇到问题？

| 错误码 | 说明 | 解决方案 |
| --- | --- | --- |
| 40164 | IP 不在白名单 | 更新 IP 白名单 |
| 40013 | AppID 无效 | 检查 AppID |
| 40001 | AppSecret 错误 | 重置 AppSecret |
| 45009 | 素材数量超限 | 删除旧素材 |
| 40007 | media_id 无效 | 重新上传图片 |
