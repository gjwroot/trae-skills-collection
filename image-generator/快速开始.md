# 🚀 生图模型 - 快速开始

## 3 步开始使用

### 第 1 步：安装依赖

```bash
cd image-generator
pip install -r requirements.txt
```

### 第 2 步：配置 API Key

复制配置文件：

```bash
cp config.example.env .env
```

编辑 `.env`，填入你的 OpenAI API Key：

```env
OPENAI_API_KEY=sk-你的OpenAI_Key_在这里
```

### 第 3 步：生成你的第一张图！

```bash
python generate_image.py "一只可爱的猫咪"
```

## 📝 常用命令

### 生成公众号封面图（推荐）

```bash
python generate_image.py \
  "一个程序员的办公桌，旁边有绿植，电脑屏幕显示 Git 分支图，温馨阳光，极简风格" \
  --size 1792x1024 \
  --prefix branch_life_cover
```

### 生成多张图

```bash
python generate_image.py "美丽的风景" --num 3
```

### 使用 DALL-E 2（更便宜）

```bash
python generate_image.py "一只小狗" --provider openai_dalle2
```

### 指定输出目录

```bash
python generate_image.py "日落" --output ./my_images
```

## 🎛️ 参数说明

| 参数 | 说明 | 默认值 |
| --- | --- | --- |
| `prompt` | （必需）图片描述 | - |
| `--provider` / `-p` | 模型提供商 | `openai_dalle3` |
| `--size` / `-s` | 图片尺寸 | `1792x1024` |
| `--style` | 风格（vivid/natural） | `vivid` |
| `--num` / `-n` | 生成数量 | `1` |
| `--output` / `-o` | 输出目录 | `./generated_images` |
| `--prefix` | 文件名前缀 | `image` |

## 🆘 常见问题

**Q: 提示 "API key not found"？**
A: 检查 `.env` 文件是否正确配置了 `OPENAI_API_KEY`

**Q: 生成失败怎么办？**
A: 检查 API Key 是否有效，账户是否有余额

**Q: 图片尺寸怎么选？**
A: 公众号封面推荐用 `1792x1024`
