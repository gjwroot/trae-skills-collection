# Overseas Mall - 海外电商小程序

一个功能完整的海外电商微信小程序，支持完整购物流程、Mock数据、搜索分享等功能。

## 项目特点

- 🛍️ **完整购物流程** - 浏览、搜索、购物车、下单
- 👤 **用户系统** - 登录注册、个人中心、订单管理
- 🔍 **智能搜索** - 搜索历史、热门搜索、搜索结果
- 📱 **商品分享** - 分享给好友、分享到朋友圈
- 🔔 **消息通知** - 个人中心消息入口
- 📦 **Mock数据** - 无需后端即可完整浏览体验
- ⚡ **性能优化** - 图片懒加载提升流畅度

## 配置说明

### 修改API地址
编辑 `utils/config.js`:
```javascript
module.exports = {
  baseUrl: 'https://your-api-domain.com',
  timeout: 10000,
  appId: 'your-app-id'
};
```

### 添加新API
编辑 `utils/api.js`:
```javascript
module.exports = {
  // 添加新的API模块
  newModule: {
    endpoint: '/new-module/endpoint'
  }
};
```

## 使用说明

1. **导入项目**
   - 下载微信开发者工具
   - 导入项目文件夹 `overseas-mall`
   - 填写AppID（测试可使用测试号）

2. **配置服务器**
   - 修改 `utils/config.js` 中的 `baseUrl`
   - 确保服务器域名已在小程序后台配置

3. **添加图标资源**
   - 在 `assets/icons/` 添加所需图标
   - 在 `assets/images/` 添加图片资源

4. **开始开发**
   - 修改页面样式和逻辑
   - 添加新功能页面
   - 对接后端API

## 注意事项

1. **登录状态**
   - 所有需要登录的操作会自动检查登录状态
   - 未登录会跳转到登录页面

2. **Token管理**
   - Token存储在本地Storage
   - 401错误会自动清除Token并跳转登录

3. **图片资源**
   - 需要自行添加图标和图片资源
   - 建议使用CDN加载图片

4. **支付功能**
   - 需要配置微信支付商户号
   - 需要后端提供支付接口
