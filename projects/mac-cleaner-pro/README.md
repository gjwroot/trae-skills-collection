# Mac 清理助手 Pro

一个真正能清理 Mac 垃圾的工具，安全、高效、可视化。

## 功能特性

- 🗑️ 清理系统缓存
- 📝 清理日志文件
- 🗑️ 清空废纸篓
- ⚡ 清理临时文件
- 📥 清理下载文件夹
- 📧 清理邮件附件
- 🔄 清理更新文件
- 📋 清理重复文件

## 技术栈

- Python 3.x
- HTML5 + CSS3 + JavaScript
- Flask (本地服务器)

## 快速开始

### 1. 安装依赖

```bash
pip install flask
```

### 2. 启动服务

```bash
python server.py
```

### 3. 打开浏览器

访问 http://localhost:5000

## 注意事项

- 本工具需要管理员权限来清理某些系统文件
- 清理前会显示详细的清理内容，确保安全
- 清理操作不可逆，请谨慎操作

## 安全保障

- 只清理安全的临时文件和缓存
- 不会删除用户数据和重要文件
- 所有清理操作都有详细的日志记录
