# 像素地下城游戏

## 项目需求

### 游戏概述
- 一个小型像素风格的地下城探险游戏
- 玩家控制角色在地下城中探索，与怪物战斗，收集物品
- 包含基本的游戏机制：移动、战斗、物品系统、关卡系统

### 技术栈
- 前端：HTML5 Canvas + JavaScript
- 样式：CSS3
- 资源：像素风格的游戏素材

### 核心功能
1. **游戏核心**
   - 玩家角色控制（上下左右移动）
   - 随机生成的地下城地图
   - 怪物AI和战斗系统
   - 物品收集和使用
   - 关卡系统和进度保存

2. **视觉效果**
   - 像素风格的游戏画面
   - 简单的动画效果
   - 游戏状态显示（生命值、物品栏等）

3. **用户体验**
   - 响应式控制（键盘和触摸支持）
   - 游戏教程和提示
   - 游戏存档和加载

## 项目结构
```
pixel-dungeon-game/
├── index.html          # 游戏主页面
├── css/
│   └── style.css       # 游戏样式
├── js/
│   ├── game.js         # 游戏核心逻辑
│   ├── player.js       # 玩家角色
│   ├── enemy.js        # 敌人AI
│   ├── map.js          # 地图生成
│   ├── items.js        # 物品系统
│   └── ui.js           # 用户界面
└── assets/
    ├── images/         # 游戏图片资源
    └── sounds/         # 游戏音效
```
